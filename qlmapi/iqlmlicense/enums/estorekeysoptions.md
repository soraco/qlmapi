# EStoreKeysOptions

| Name                        |                     Description                    | Value |
| --------------------------- | :------------------------------------------------: | ----- |
| EStoreKeysPerUser           |                 store keys per user                | 1     |
| EStoreKeysPerMachine        |               store keys per machine               | 2     |
| EStoreKeysPerUserAndMachine | store keys at the user level and the machine level | 3     |
