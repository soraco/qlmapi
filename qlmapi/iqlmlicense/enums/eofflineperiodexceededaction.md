# EOfflinePeriodExceededAction

| Name           |         Description         | Value |
| -------------- | :-------------------------: | ----- |
| DoNothing      | No action will be performed | -1    |
| ExitApp        |    Exits the application    | 0     |
| FireEvent      |        Fires an event       | 1     |
| ThrowException |     Throws an exception     | 2     |
